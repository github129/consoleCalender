# ConsoleCalender.sln

コンソールにカレンダーを表示するソリューション

# システム要件

* Windowsコマンドライン


# What is this?

コンソールに指定した年月のカレンダーを表示します。
特に指定がない場合は当月のカレンダーを表示します。


# 主な機能
* １か月単位のカレンダーの表示
* オプションとして[-s][-m][-h][-n]が用意されています。
* -sを利用することでカレンダーを日曜日始まりにすることができます。（デフォルトは日曜始まり）
* -mを利用することでカレンダーを月曜日始まりにすることができます。
* -hを利用することで当月のハイライト機能を削除することができます。
* -nの後に整数を入力することで、その数と同じだけのカレンダーを生成します。例) -n 3 とすると指定した年月から１月前＋１ヵ月を生成します。
* 引数を入力しない場合は当年月を生成します。
* 引数として月を入力しない場合その入力した年の１２ヵ月分のカレンダーを生成します。

## 基本的な引数の入力方法

* 年　月　の順で入力します。
  月は年が入力されていない場合は判定されません。

* 年　オプション　月　のような入力も可能です。
